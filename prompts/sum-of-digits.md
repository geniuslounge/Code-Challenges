---
title: Sum of Digits
difficulty: Easy
---

Write a function that takes a positive integer and returns the sum of its digits.

```
sum_of_digits(493)    →  16
sum_of_digits(1000)   →  1
sum_of_digits(99)     →  18
```

**Part 2** — Write a function that repeatedly sums the digits until a single digit remains (the "digital root").

```
digital_root(493)  →  7   (4+9+3=16, 1+6=7)
digital_root(99)   →  9   (9+9=18, 1+8=9)
```
