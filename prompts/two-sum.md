---
title: Two Sum
difficulty: Medium
---

Given a list of integers and a target value, find the two numbers that add up to the target and return their indices.

You may assume there is exactly one solution, and you may not use the same element twice.

```
two_sum([2, 7, 11, 15], 9)   →  [0, 1]  (2 + 7 = 9)
two_sum([3, 2, 4], 6)        →  [1, 2]  (2 + 4 = 6)
two_sum([1, 5, 3, 7, 2], 10) →  [1, 3]  (5 + 7 = 10... wait, is it? Check your work!)
```

**Part 2** — What's the most efficient solution? Can you do it in a single pass through the list?
