# Security Policy

## Scope

This repository contains only markdown files with coding challenge descriptions. There is no executable code, no authentication, and no user data — the attack surface is minimal.

That said, we take the integrity of this project seriously.

## Reporting a Vulnerability

If you discover a security issue (e.g. a malicious file, a compromised dependency in the GitHub Actions workflow, or a supply chain concern), please **do not open a public issue**.

Instead, report it privately using [GitHub's private vulnerability reporting](https://github.com/geniuslounge/Code-Challenges/security/advisories/new) or email the maintainers directly.

We will respond within 72 hours.
